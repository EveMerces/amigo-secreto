# 🎁 Amigo Secreto  

Um projeto simples de sorteio de amigo secreto feito com **HTML, CSS e JavaScript**.  

## 🚀 Funcionalidades  
- Adicione nomes à lista de participantes  
- Realize o sorteio automaticamente  
- Interface intuitiva e fácil de usar
